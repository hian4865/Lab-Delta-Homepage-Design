# Lab-Delta-Homepage-Design

CIS-209 / Spring 2026 - Solo Project (Business Name: Elephant Bookstore)

<h1>Elephant Bookstore Description</h1>

Elephant Bookstore - Located in Birmingham Alabama, (Est. 2004)

Elephant Bookstore, a locally owned and entirely independently owned bookstore, was established in 2004 to offer an affirming space to people of all kinds. Any sort of book worm, teacher, student, and enthusiast could find respice in the halls of this library and lose themselves in the stores curated collection of books. From timeless classics to bestselling novels, anyone could find and enjoy a book on any topic they could find. Ever since Elephant Bookstore was first established it has been a warm and welcoming place for the community as a whole and serves as a place for connection and growth within the community.

1. Project Website structure will be built using HTML5.
   
2. Project Website layout and styling will be built with CSS (e.g. Flexbox and CSS Grid.)
   
3. Project Website user interactive features will be using JavaScript.
   
4. The project will be hosted using GitHub Pages.
   
5. No external APIs will be required at this stage.
    
6. All website versions will be maintained and controlled via Github.
